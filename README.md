# rpy-translator

## Overview

`rpy-translator` is a tool designed to translate text and strings in RP (Ren'Py) games efficiently, supporting multiple languages and a variety of file formats.

## Features

- **Multi-language Support**: Easily add translations for various languages.
- **File Format Flexibility**: Supports multiple formats for text files commonly used in RP games.
- **User-friendly Interface**: Simple command-line interface for ease of use.

## Installation

To install `rpy-translator`, simply clone the repository and install the dependencies:

```bash
git clone https://github.com/xzed78-hash/rpy-translator.git
cd rpy-translator
pip install -r requirements.txt
```

## Usage

### Translating a File

To translate a file, run the following command:

```bash
python translate.py <input_file> <output_file> --language <language_code>
```

- `<input_file>`: The file you want to translate.
- `<output_file>`: The file where the translated text will be saved.
- `<language_code>`: The code of the language you want to translate to (e.g., `en`, `es`, `fr`).

### Supported Languages

- English (en)
- Spanish (es)
- French (fr)
- German (de)
